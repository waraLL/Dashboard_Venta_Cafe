# Dashboard de Ventas y Comportamiento de Clientes - Power BI ☕

## Objetivo
Analizar el rendimiento de ventas y el comportamiento de clientes en una cafetería utilizando Power BI. 
- Identificando patrones de consumo
- Productos más demandados
- Horarios de mayor actividad

## KPIs
- Ventas totales
- Ticket promedio
- Clientes únicos
- Frecuencia de compra
- Hora pico
- Total de transacciones

## Hallazgos principales
- La mayoría de transacciones se realizan con tarjeta.
- Los clientes recurrentes generan el mayor volumen de ventas.


Fuente de datos, https://www.kaggle.com/datasets/reignrichard/coffee-store-sales
